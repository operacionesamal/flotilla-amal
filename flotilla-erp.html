Instrucciones de despliegue:

4. Exporta el JSON regularmente como respaldo.
5. Para acceso desde cualquier lugar, sube el archivo a GitHub Pages (gratis).
6. Para múltiples usuarios, integra Firebase Firestore reemplazando el objeto store.
